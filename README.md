ErgoNext – Ergonomic Study Desk Design
Overview
ErgoNext is a Design Thinking project focused on improving student posture through an ergonomically designed study desk. The project combines principles of art (Dots, Lines, and Shapes) with functional design to create a workspace that enhances comfort, productivity, and physical well-being.
This web application presents the full journey of the project, including research, concept development, specifications, prototype visualization, process flow, and user feedback collection.

Features
1. Navigation System


Smooth tab-based navigation


Sections include: Home, Research, Concept, Specifications, Prototype, Process Flow, and Testing Data


Active tab highlighting for better user experience


2. Home Section


Introduction to the ErgoNext project


Highlights the integration of geometry with ergonomic design


Call-to-action buttons for quick navigation


3. Research Section


Interactive image comparison slider


Demonstrates difference between poor posture and ergonomic posture


Helps users visually understand the problem and solution


4. Concept Section
Explains design principles based on:


Dot: Connection points and focus areas


Line: Movement, posture alignment, and adjustability


Shape: Structural stability and surface design


5. Specifications Section
Details about:


Desk dimensions and adjustability


Ergonomic enhancements


Materials used (bamboo, carbon fiber, aluminum)


Connectivity features like USB-C ports and wireless charging


6. Prototype Section


Visual representation of the desk


Interactive hotspots explaining key features


Highlights real-world application of the concept


7. Process Flow Section


Uses Mermaid.js to display Design Thinking stages:


Empathize


Define


Ideate


Prototype


Test




8. Feedback Section


User feedback form


Collects:


Name


Email/Student ID


Rating


Comments




Displays success message after submission


9. Theme Toggle


Light/Dark mode switch for better accessibility
